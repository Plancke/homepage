---
title: Bookmark
description: Bookmark Information Widget Configuration
---

This allows you to add a bookmark, allows displaying [service icons](../../configs/services.md#icons).

```yaml
- bookmark:
    icon: homepage
    href: https://gethomepage.dev
    text: "bookmarks!"
    text_size: "sm"
```
